LU3IN003 - Un projet de tomographie discrète.

Vous trouverez dans ce répertoire 6 fichiers .py et un répertoire. 

- Le répertoire “instances”: contient les instances 1.txt à 16.txt fournies (et 17.txt, celle utilisée dans la page de garde).
- Le fichier “colorierCase.py”: contient les réponses aux questions 4 et 7 (qui consistent à tester s’il existe un coloriage possible d’une ligne ou d’une colonne avec une séquence donnée).
- Le fichier “constantes.py”: renferme les constantes utilisées.
- Le fichier "grille.py": comporte les fonctions concernant la grille (tableau à 2 dimensions).
- Le fichier “lecture.py”: renferme la fonction de conversion en grille des instances.txt (question 9).
- Le fichier “resolution.py”: contient les algorithmes de propagation et d’énumération (réponses aux questions 9 et 13).
- Le fichier “main.py”: comporte les tests effectués.
